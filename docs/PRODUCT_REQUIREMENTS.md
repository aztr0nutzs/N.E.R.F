# Product Requirements (NERF)

NERF = Network Explorer & Reporting Framework

## Platforms
1) Android (primary)
2) Web (secondary)

## Required screens
- Speedtest
- Network Map
- Devices
- Analytics
- Settings

## Theme System (MANDATORY)
- Multiple themes must be supported.
- Must include:
  - Native theme: `neon_nerf`
  - HTML packs: `speedtest6`, `nerf_speed2`
- Theme selection must persist across app restarts.

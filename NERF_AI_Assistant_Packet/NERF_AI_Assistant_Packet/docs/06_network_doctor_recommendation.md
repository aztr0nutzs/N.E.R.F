# Standout Feature Recommendation: Network Doctor

The assistant is not the only missing feature. The strongest companion feature is a structured diagnosis and remediation center called **Network Doctor**.

## Why it matters

Most network apps show:
- raw data
- router settings
- device lists
- fake intelligence

Very few do this well:
- identify what is wrong
- explain why
- recommend what to do next
- separate safe one-tap actions from manual steps

## Network Doctor should do

- continuously evaluate health
- surface ranked issues
- explain likely cause
- estimate impact
- offer one-tap actions
- provide step-by-step fixes for manual issues

## Relationship to Assistant

- Assistant = conversational interface
- Network Doctor = structured expert diagnosis screen

These should share:
- AssistantContextSnapshot
- NetworkDiagnosisEngine
- RecommendationEngine

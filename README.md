# NERF (Network Explorer & Reporting Framework)

Android-first network control tool. Web app second.

## Included
- Jetpack Compose multi-screen scaffold (Speedtest / Map / Devices / Analytics / Settings / Preview)
- Theme system is **mandatory**:
  - Native Compose theme (`neon_nerf`)
  - HTML theme packs in `app/src/main/assets/themes/*` (for web parity + in-app WebView preview)

## Theme packs (DO NOT REMOVE)
- `app/src/main/assets/themes/speedtest6/index.html`
- `app/src/main/assets/themes/nerf_speed2/index.html`
